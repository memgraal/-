# Система управления сетью спортивных залов

## Описание

Проект **ProjectMaxitet** представляет собой базу данных для управления сетью спортивных залов.  
Система позволяет хранить и обрабатывать информацию о:

- Спортивных залах (адрес, вместимость)  
- Клиентах (данные, регистрация, привязка к залу)  
- Тренерах (данные, специализация, привязка к залу)  
- Абонементах (тип, длительность, стоимость)  
- Занятиях (классы, расписание, тренер)  
- Записях клиентов на занятия  
- Подписках клиентов на абонементы  
- Спортивном инвентаре залов  
- Мероприятиях, проводимых в залах  

Проект реализован на **MySQL/MariaDB** и включает тестовые данные для демонстрации связей между таблицами.

---

## Структура базы данных

### Таблицы

1. **SportsHalls** – информация о спортивных залах.  
2. **Clients** – информация о клиентах и привязка к залу.  
3. **Trainers** – информация о тренерах и привязка к залу.  
4. **Memberships** – информация о типах абонементов.  
5. **Classes** – информация о занятиях и назначенных тренерах.  
6. **ClientClasses** – записи клиентов на занятия.  
7. **ClientMemberships** – подписки клиентов на абонементы.  
8. **SportsEquipment** – спортивный инвентарь залов.  
9. **Events** – мероприятия, проводимые в залах.
